## 🛠 Tech Stack & Features

엔진의 최신 피처를 최대한 활용하고 데이터 지향적 설계(DOD)를 따르는데 초점을 맞췄습니다.

### Core Systems
- **GAS (Gameplay Ability System)**: 어빌리티 및 상태 관리
- **Enhanced Input**: 런타임 입력 컨텍스트 관리
- **Mover 2.0**: 차세대 캐릭터 무브먼트 시스템
- **SmartObject**: 환경 기반 상호작용

### Performance & Scalability
- **MassEntity**: 수천 개 개체 제어를 위한 데이터 지향 아키텍처
- **StateTree**: 계층형 상태 머신 기반 로직 처리
- **Custom Physics**: Mass Processor를 활용한 커스텀 중력 및 물리 계산

### World & Tools
- **World Partition**: 대규모 맵 스트리밍 및 로딩 최적화
- **PCG (Procedural Content Generation)**: 절차적 환경 생성
- **Geometry Script**: 블루프린트를 통한 메쉬 변형
