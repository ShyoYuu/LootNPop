# 어빌리티 시스템 - 게임 기획서

## 1. 개요

LootNPop의 전투는 **GAS(Gameplay Ability System)** 기반으로 동작한다. 플레이어는 "맨손" 무기를 기본 장착한 상태로 시작하며, LootPod에서 획득한 아이템을 장착·보유함에 따라 어빌리티와 스텟이 강화.

핵심 게임 루프:

```
LootPod 루팅 → 아이템 획득 → 장착/보유 → 어빌리티/스텟 강화 → 더 강한 전투
```

---

## 2. 어빌리티 획득 구조

| 아이템 타입 | 슬롯 수 | GAS 연동 |
|:---|:---|:---|
| **무기 (Weapon)** | 1개 | 무기 GA (기본 공격 포함) + 무기 스텟 GE |
| **스킬 - Active** | N개 (Config, 기본 4) | 키 입력으로 발동하는 스킬 GA |
| **스킬 - Passive** | 무제한 | 조건 충족 시 자동 발동 스킬 GA |
| **버프 (Buff)** | 무제한 | 스텟 GE (기간제) |

---

## 3. 아이템 타입별 상세 기획

### 3.1 무기 아이템 (Weapon)

무기를 장착하면 해당 무기의 **고유 기본 공격 어빌리티**가 활성화된다. 무기 종류에 따라 공격 방식이 전혀 달라지는 것이 핵심.

- **슬롯**: 단 하나의 무기 슬롯. 새 무기 장착 시 기존 무기는 인벤토리로 반환.
- **기본 무기 "맨손"**: 게임 시작 시 맨손 무기가 기본 장착된 상태. 맨손도 하나의 무기 아이템으로 취급하여 시스템 규칙이 일관됨.
- **기본 공격 GA**: 무기마다 고유한 기본 공격 GA를 보유. 무기 교체 시 이전 무기의 기본 공격 GA는 제거되고 새 무기의 GA가 부여됨.
- **추가 어빌리티**: 무기에 따라 기본 공격 외 추가 GA를 보유할 수 있음. (예: 검 → 차지 공격 GA)
- **무기 스텟**: 공격력·공격 속도·사거리 등의 스텟이 GE로 적용됨.

| 무기 예시 | 기본 공격 특성 | 추가 GA |
|:---|:---|:---|
| 맨손 | 근접, 짧은 사거리, 빠른 속도 | — |
| 거대 해머 | 근접, 광역 판정, 느린 속도 | 지진 강타 |
| 고무공 발사기 | 원거리, 튀어오름 판정 | 연속 발사 |
| 중력 건 | 원거리, 적 끌어당기기 | 중력장 투사 |

### 3.2 스킬 아이템 (Skill)

발동 방식에 따라 두 종류로 나뉜다.

#### Active Skill (키 맵핑)
- 플레이어가 지정된 키를 눌러 **직접 발동**하는 어빌리티.
- 슬롯 제한 있음. (기본 4개, Project Settings에서 조절)
- 슬롯이 꽉 찬 상태에서 새 Active Skill을 장착하면 기존 슬롯 중 하나와 교체.
- 예) 순간이동, 광역 넉백, 보호막 전개

#### Passive Skill (자동 발동)
- 특정 조건이 충족되면 **자동으로 발동**.
- 슬롯 제한 없음.
- 예) 피격 시 순간 무적, 체력 30% 이하에서 이동속도 증가

### 3.3 버프 아이템 (Buff)

버프 아이템은 **기간제**다. 아이템에는 최대 지속 시간과 현재 남은 시간이 저장되며, **플레이어가 보유하고 있는 동안에만 시간이 차감**된다.

| 상황 | 남은 시간 변화 |
|:---|:---|
| 인벤토리에 보유 중 | 실시간 차감 |
| 바닥에 드랍됨 | **차감 중지** |
| 다른 플레이어가 주움 | 다시 차감 재개 |
| 남은 시간 0 도달 | 아이템 소멸, 스텟 효과 제거 |

> 예시: 5분짜리 버프 아이템을 4분간 보유 후 드랍 → 바닥에 1분짜리 아이템으로 존재. 다른 플레이어가 주우면 1분간 버프 효과 적용 후 소멸.

## 4. 인벤토리 구조

플레이어의 어빌리티 셋과 아이템은 모두 **PlayerState에 귀속**된다. 폰이 일시적으로 교체되는 상황(기간제 폰 아이템 등)에도 스킬·버프 효과가 유지되어야 하기 때문.

### 4.1 PlayerState 귀속

| 저장소 | 내용 | 개수 |
|:---|:---|:---|
| 무기 슬롯 | 현재 장착 무기 | 1 |
| Active Skill 슬롯 | 키 맵핑 스킬 | N개 (Config) |
| Passive Skill 목록 | 자동 발동 스킬 | 무제한 |
| 버프 아이템 목록 | 기간제 스텟 버프 | 무제한 |
| 아이템 보관함 | 미장착 아이템 | 무제한 |

> 버프 아이템과 Passive Skill은 장착/해제 개념 없이 인벤토리에 보유하는 것만으로 자동 활성화.

## 5. 스텟 시스템

| 스텟 | 설명 | 버프 아이템 영향 |
|:---|:---|:---|
| Health / MaxHealth | 현재/최대 체력 | MaxHealth 증가 가능 |
| AttackPower | 기본 공격 및 어빌리티 피해 배율 | ✅ |
| AttackSpeed | 공격 속도 배율 (애님 재생 속도와 연동) | ✅ |
| DefensePower | 피격 시 피해 감소 | ✅ |
| MoveSpeed | 이동 속도 배율 | ✅ |
