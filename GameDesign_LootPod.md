# LootPod System - 게임 기획서

## 1. 개요

**LootPod**은 LootNPop 월드 전역에 배치되는 핵심 상호작용 오브젝트. 플레이어는 적들의 방해를 뚫고 Pod를 루팅하여 강력한 무기·버프·스킬을 획득하고, 이를 써서 적들을 날려버리는 것이 게임의 핵심 루프.

---

## 2. 핵심 루프: "Find → Loot → Pop!"

### 2.1 발견 (Find)

- 월드 곳곳에 무작위로 스폰되며, **빛기둥(Light Pillar)** 이펙트로 원거리에서도 식별 가능.
- 메인 내벽과 부유 행성 외부 표면 모두에 배치.

### 2.2 루팅 (Loot)

1. 플레이어가 접근하여 상호작용 입력(F키)을 누르면 바닥에 **루팅 구역(Looting Zone)** 홀로그램 링이 표시되고 게이지가 차오름.
2. 루팅 구역 안에 머물며 게이지를 채워야 함.

**취소 조건:**

| 조건 | 결과 |
|:---|:---|
| 루팅 구역 이탈 | 즉시 취소, 게이지 초기화 |
| 적에게 피격 (Hit 발생) | 즉시 취소, 게이지 초기화 |

> 피격 취소는 HitDetection 시스템과 연동. 피격 판정 시 플레이어 엔티티의 `FLNPPlayerLootingTag`가 제거되고, 다음 프레임 루팅 프로세서가 Idle로 복귀. (→ `TechDesign_LootPod.md`, `TechDesign_HitDetection.md`)

**방해 요소:** 주변의 적 NPC는 루팅 중인 플레이어를 최우선 타겟으로 지정.

### 2.3 획득 (Power up)

- 게이지가 완료되면 LootPod이 파열(`Popped`)되며 보상이 월드에 스폰.
- 보상은 즉시 적용(버프)되거나 픽업 가능한 드랍 아이템으로 출현.

---

## 3. 보상 유형

| 유형 | 예시 | 비고 |
|:---|:---|:---|
| **Weapons** | 거대 해머, 고무공 발사기, 중력 건 | 교체 가능한 장비 |
| **Buffs** | 이동 속도 증가, 캐릭터 거대화, 점프력 강화 | 지속 시간 존재 |
| **Skills** | 순간이동, 광역 넉백, 보호막 | 쿨타임 기반 액티브 스킬 |
| **Items** | 회복 키트, 함정 설치기, 일회용 투척물 | 소모성 아이템 |
| **Gimmicks** | 고정형 터렛 탑승권, 점프 패드 활성화 | 환경 상호작용 |

보상 종류는 `PodID`를 통해 외부 데이터 에셋으로 관리. 코드 수정 없이 보상 유형을 추가·변경 가능.

## 4. 연출 (VFX/SFX)

| 상태 | 연출 |
|:---|:---|
| 대기 | 하늘 높이 솟은 빛기둥, 풍선처럼 부풀어 오르는 루프 애니메이션 |
| 루팅 중 | 바닥 홀로그램 링, Pod 내부 소용돌이 에너지 이펙트 |
| 루팅 취소 | '푸쉬쉭-' 하는 김빠지는 사운드와 함께 작아지는 연출 |
| 루팅 완료 | 화려한 색종이(Confetti)와 함께 보상이 튀어나오는 연출 |

---

## 5. 밸런스 및 난이도 연계

- 월드에 남은 LootPod 수가 줄어들수록 적 NPC가 점진적으로 강화. (→ `GameDesign_EnemyNPC.md`)
- 특정 구역의 Pod가 모두 소모되면 일정 시간 후 새로운 위치에 리스폰.

---
